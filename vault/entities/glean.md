# Glean

Enterprise search company. In the [[trellis|Trellis]] competitive matrix, Glean is the cleanest **structural** comparison: same problem space (find the answer the firm already knows), incompatible architecture.

## Key facts

- **Category**: enterprise search (and now generative answers over indexed corpora). (see [[trellis-product-brief]])
- **Why it doesn't fit law firms**: Glean's cloud-first model is **incompatible with most firm policies** around [[attorney-client-privilege|privilege]]. Trellis is **privacy-architected from day one for privilege**.
- **Adjacent**: Hebbia (enterprise search / extraction-oriented).

## Relations

- **Compares with**: [[trellis]] on problem space; differs sharply on architecture
- **Adjacent**: Hebbia, [[notion]] (generic team knowledge)

## Open questions

- Has Glean shipped a single-tenant on-prem option that would satisfy firm privilege concerns? If yes, the architectural distinction narrows.
