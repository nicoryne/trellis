# Performance-review integration — does it ever ship?

## Status

**Open** — awaiting Keith's answer.

## Why it matters

The earlier five-layer adoption model included **Career Capital** (attribution, citations, performance-review integration). That layer was rejected as too soft to carry strategy, but the idea reappears as a "soft consideration." Either it ships as a real feature or it stays as positioning — leaving it ambiguous risks customer disappointment ("you said this would help with reviews"). (see [[adoption-strategy]] §"What was rejected")

## What we know so far

- The earlier model proposed performance-review tie-in.
- Rejected as not robust enough to drive adoption by itself.
- Still mentioned as a soft reinforcement.
- No PM-review integration is on any roadmap.

## What would resolve it

- A clear "no, not on roadmap" or "yes, V2 feature, here's the rough shape."
- Decide whether published-insight authorship attribution surfaces in any UI ("Jane authored 12 of the firm's top-cited insights this quarter").
- Confirm what audit data would be needed to support such a feature (mostly already in V1 audit log).
