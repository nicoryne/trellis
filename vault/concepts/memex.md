# Memex

Vannevar Bush's 1945 vision of a personal, curated knowledge store with **associative trails** between documents. The spiritual predecessor of the [[llm-wiki-pattern]].

## Definition

In Bush's essay *As We May Think* (1945), the Memex is a hypothetical device — imagined as a desk with microfilm storage — where a single user accumulates documents and constructs explicit links ("trails") between them, then revisits and shares those trails.

## Key claims

- The Memex was **private, actively curated**, with the connections between documents as valuable as the documents themselves. (see [[karpathy-llm-wiki-pattern]])
- Bush's vision was closer to this than to what the web actually became (public, automatically indexed, unstructured by default).
- **The part Bush couldn't solve** was *who does the maintenance* of the trails. The LLM Wiki pattern resolves it: the LLM does.

## Contrast with adjacent concepts

- **Not the same as the web** — the web optimized for publishing and discovery; the Memex optimizes for the individual reader's reasoning.
- **Different from [[rag]]** — Memex is a structured, persistent artifact; RAG is on-the-fly retrieval.
- **The [[llm-wiki-pattern]] is essentially Memex with an automated maintainer.**
